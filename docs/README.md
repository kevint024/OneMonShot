# OneMonShot Documentation

This directory contains documentation and assets for the OneMonShot project.

## Files

- `screenshot-main.png` - Main application screenshot for README
- `user-guide.md` - Detailed user guide
- `developer-guide.md` - Development setup and contribution guide
- `build-guide.md` - Build and deployment instructions

## Screenshots

Screenshots should be:
- High resolution (at least 1920x1080 source)
- Show the application in a realistic usage scenario
- Include multiple monitors when demonstrating multi-monitor features
- Use consistent Windows theme (preferably Windows 11 light theme)

## Documentation Standards

- Use Markdown format
- Include table of contents for longer documents
- Add screenshots and diagrams where helpful
- Keep language clear and beginner-friendly
- Update version numbers and dates when content changes

## Contributing to Documentation

When contributing documentation:
1. Follow the existing style and format
2. Test all code examples and procedures
3. Update relevant screenshots if UI changes
4. Ensure all links work correctly
5. Spell check and proofread content
